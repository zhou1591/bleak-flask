# python bleak 包 实现得蓝牙


## 优点

- 可以突破 20mtu 限制

- 不需要 noble 和bleno 这两个node 包一样需要覆盖通用蓝牙驱动

## 缺点

- 慢

- 读写读大概50毫秒一次 写100-150毫秒  根据数据量大小波动   webble 20毫秒  不能扫描bt 蓝牙

## 注意

- app.py 连接时候报错 无需理会 前端不拦截就好  with 里边有上下文的断开和连接 为了保持连接断开改成了 pass 不想报错可自行更改 或者读写不频繁的场景每次连接
